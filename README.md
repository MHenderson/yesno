# yesno
R wrapper for https://yesno.wtf (Yes/No as a service).

```r
drat::addRepo("MHenderson")
install.packages("yesno")
```

```r
> yesno::yesno()
$answer
[1] "no"

$forced
[1] FALSE

$image
[1] "https://www.yesno.wtf/assets/no/16-b66d2db7543f5259c86abc166d6901cf.gif"
```
